# Ejercicios de Condicionales en JavaScript

## Descripción

Este repositorio contiene una serie de ejercicios realizados para practicar el uso de estructuras condicionales en JavaScript.

Los ejercicios fueron desarrollados como parte de las actividades del curso RollingCode, utilizando conceptos básicos del lenguaje.

## Temas trabajados

- if
- else
- else if
- Operadores lógicos
- Comparaciones
- prompt()
- alert()
- Variables
- Number()

## Contenido

Se resolvieron ejercicios como:

- Número positivo o negativo
- Edad para votar
- Número par o impar
- Contraseña
- Mayor de tres números
- Calculadora básica
- Año bisiesto
- Clasificación de calificaciones
- Precio con descuento
- Verificación de acceso
- Clasificador de edades
- Validación de fechas
- Acceso por roles
- Impuestos según ingreso
- Conversión de temperatura
- Juego de adivinanza
- Evaluación de contraseña
- Tarifa por distancia
- Clasificación de películas
- Simulación de semáforo
